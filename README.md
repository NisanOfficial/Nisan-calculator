# Simple Calculator

A beautiful and functional calculator built with HTML, CSS, and JavaScript. Perfect for basic arithmetic operations with a modern user interface.

## Features

✨ **Core Functionality:**
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔘 Decimal support
- 🗑️ Delete last digit
- 🔄 Clear all (C button)

🎨 **Design:**
- Modern purple gradient background
- Responsive design (works on mobile and desktop)
- Smooth button animations and hover effects
- Clean and intuitive user interface

⌨️ **Keyboard Support:**
- **0-9** - Number input
- **+, -, *, /** - Operators
- **Enter** - Calculate result
- **Backspace** - Delete last digit
- **Escape** - Clear display

🛡️ **Safety Features:**
- Division by zero protection
- Input validation
- Error handling

## Files

- `index.html` - Calculator HTML structure
- `style.css` - Styling and responsive design
- `script.js` - Calculator logic and functionality

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start calculating!

### Using the Calculator:
- Click buttons to input numbers and operations
- Press `=` or `Enter` to get the result
- Use `C` to clear everything
- Use `DEL` or `Backspace` to remove the last digit

## Example Calculations

- `5 + 3` → `8`
- `10 * 2.5` → `25`
- `100 / 4` → `25`
- `7 - 9` → `-2`

## Browser Compatibility

Works on all modern browsers that support:
- HTML5
- CSS3
- ES6 JavaScript
