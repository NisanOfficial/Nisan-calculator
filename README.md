# Simple Calculator

A beautiful and functional calculator built with HTML, CSS, and JavaScript. Perfect for basic arithmetic operations with a modern user interface.

## Features

✨ **Core Functionality:**
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔘 Decimal support
- 🗑️ Delete last digit
- 🔄 Clear all (C button)

🎨 **Design:**
- Modern purple gradient background
- Responsive design (works on mobile and desktop)
- Smooth button animations and hover effects
- Clean and intuitive user interface

⌨️ **Keyboard Support:**
- **0-9** - Number input
- **+, -, *, /** - Operators
- **Enter** - Calculate result
- **Backspace** - Delete last digit
- **Escape** - Clear display

🛡️ **Safety Features:**
- Division by zero protection
- Input validation
- Error handling

## Files

- `index.html` - Calculator HTML structure
- `style.css` - Styling and responsive design
- `script.js` - Calculator logic and functionality

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start calculating!

### Using the Calculator:
- Click buttons to input numbers and operations
- Press `=` or `Enter` to get the result
- Use `C` to clear everything
- Use `DEL` or `Backspace` to remove the last digit

## Example Calculations

- `5 + 3` → `8`
- `10 * 2.5` → `25`
- `100 / 4` → `25`
- `7 - 9` → `-2`

## Browser Compatibility

Works on all modern browsers that support:
- HTML5
- CSS3
- ES6 JavaScript

## 🌐 Hosting & Deployment

### Option 1: GitHub Pages (Free & Easy) ⭐
1. Go to your repository settings
2. Scroll to "GitHub Pages" section
3. Select "main" branch as source
4. Your calculator will be live at: `https://NisanOfficial.github.io/Nisan-calculator/`

### Option 2: Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Click "New site from Git"
3. Connect your GitHub repository
4. Deploy automatically on every push
5. Get a live URL in minutes

### Option 3: Vercel (Free)
1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repository
3. Deploy with one click
4. Get automatic deployments on push

### Option 4: Firebase Hosting (Free)
1. Install Firebase CLI: `npm install -g firebase-tools`
2. Run: `firebase init`
3. Select "Hosting" option
4. Deploy: `firebase deploy`

### Option 5: Traditional Web Hosting
- Upload files via FTP to any web hosting provider
- Access via your domain name
